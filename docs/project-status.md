- ✅ 관리자 인증 기반 수정/삭제

## 기술 스택
- **프레임워크**: Next.js 15 (App Router)
- **스타일링**: Tailwind CSS v4
- **UI 컴포넌트**: shadcn/ui
- **아이콘**: Lucide React
- **데이터 저장**: localStorage
- **배포**: Vercel
- **PWA**: Service Worker
